# Kivo Judge Walkthrough (90 seconds)

1. Sign in with Google.
2. First-time users are routed into the guided demo automatically.
3. In demo, run: Normalize -> Build Analytics -> Simulate Premium Unlock.
4. Finish demo and land in Dashboard.
5. Connect an App Store source in Data Sources.
6. Trigger `Sync` and show sync telemetry (pages, fetched, inserted, duplicates).
7. Return to Dashboard and highlight Top 3 Revenue Opportunities with evidence and confidence.
8. Click a locked premium opportunity (free plan) to show contextual upgrade modal.
9. Run AI Intelligence and show usage meter / trial gating.
10. In Data Sources, open webhook card and show workspace API key security + ingestion URL.

## Key talking points

- Data trust: paginated App Store backfill, dedupe, incremental sync, telemetry.
- Security: per-workspace webhook API key + app ownership checks.
- Decision UX: evidence-backed opportunities tied to revenue impact.
- Monetization: balanced free value with contextual premium conversion points.
- Lingo.dev: translation fidelity powers cross-market comparability and localized actions.
